An embedded system is a small computer designed to perform one specific task.
It is made of embedded hardware (microcontroller, sensors, components) and embedded firmware (program written in C).

It is used in devices like washing machines, mobiles, cars, AC, and medical equipment.
To learn embedded systems, a student should know basic electronics, electronic components, measuring instruments, and microcontroller programming.
